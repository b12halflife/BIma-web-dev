# euler project | Soal 1

1.28.2023

Di euler project memiliki soal sebagai berikut "If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23. Find the sum of all the multiples of 3 or 5 below 1000."

Untuk menyelesaikan masalah ini saya akan mengunakan python. Dari soal di atas hanya kelipatan 3 dan 5 saja yang akan di tambahkan. Dengan python kita bisa mengunakan if dan operator logika (or) sebagai dasar dari code dan menggunakan operator modulo % untuk memeriksa apakah suatu bilangan habis dibagi 3 atau 5 dan akan di tambahkan di variabel temp. 

              temp = 0 
                if i % 3 == 0 or i % 5 == 0: 
                temp += i

Kondisi sudah selesai, sekarang kita lanjutkan ke langkah selanjutnya yaitu perulangan untuk menyelesaikan masalah di atas, sebagai berikut : 

              tamp = 0 
              for i in range(0,1000):
                if i % 3 == 0 or i % 5 == 0: 
                tamp += i

              print(tamp)

Jika berhasil akan di dapat hasil tamp yaitu 233168

Kesimpulan : Untuk memecahkan masalah ini, anda dapat menggunakan sebuah algoritma sederhana yang iterates atas semua angka kurang dari 1000,dan cek jika mereka dibagi oleh 3 atau 5 = 0 dan jika demikian, menambahkan mereka ke Tamp.

Dengan menggunakan perulangan for dan pengkondisian if sebagai dasar dan menggunakan operator modulo % untuk memeriksa apakah suatu bilangan habis dibagi 3 atau 5 dan akan di tambahkan di variabel temp.
